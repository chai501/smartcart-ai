# Data Science Layer

This folder contains Python scripts for analytics and recommendations.

## Stack

- Python (Pandas, NumPy)
- scikit-learn

## Setup

1. (Recommended) Create a virtual environment:
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
