# Smart E-Commerce Analytics & Recommendation System

## Overview

A full-stack platform where:

- Users browse products
- System tracks behavior (clicks, purchases)
- Analytics dashboard for insights
- Product recommendations using ML

## Stack

- **Frontend:** React (Vite), Tailwind CSS, Axios, Chart.js/Recharts
- **Backend:** Node.js + Express, REST APIs, JWT Auth
- **Database:** MongoDB
- **Data Science:** Python (Pandas, NumPy, scikit-learn)

## Structure

- `frontend/` — React app
- `backend/` — Express API
- `data-science/` — Python scripts for analytics/recommendations
